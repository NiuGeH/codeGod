## IconFont使用方法

FrosUI内部提供了基于阿里Iconfont的矢量图标，存放路径在assets/font/addicon中，目前默认自带的一套icon是菜鸟物流的icon，使用者可以根据自身需要在Iconfont上面创建项目并导入需要的icon，下载后替换现有的icon库即可。对于iconfont的使用方法如下：

```html

<!-- 使用样式 注意需要带上iconfont -->
<span class="iconfont fros-icon--guoji"></span>

<!-- 直接使用unicode -->
<span>&amp;#xe6ec;</span>

```
